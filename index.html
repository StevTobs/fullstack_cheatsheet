    HTML

    Fundamental HTML

    CODE DESCRIPTION
    <a> This element creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
    < a href="https://example.com">Website </ a >

    HTML Table Reference

    CODE DESCRIPTION
    <table> The element used to create a table.
    <tr> The element used to add rows to a table.
    <colspan> Table data can span rows using this attribute.
    <thead>, <tbody>, <tfoot> Tables can be split into three main sections: a head, a body, and a footer.

    Link States

    CODE DESCRIPTION
    :link (e.g., <a:link >) Matches links that have not yet been visited.
    :hover Matches when a user designates an item with a pointing device, such as holding the mouse pointer over the item.
    :active Matches when an item is being activated by the user. For example, when the item is clicked on.
    :visited It applies once the link has been visited by the user.
